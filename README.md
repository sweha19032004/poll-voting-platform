Online Poll & Voting Platform
A simple real-time polling platform for creating and participating in quick surveys.
**Technology Stack**
Frontend: React.js
Backend: Express.js

**Features**
Create polls - Design polls with multiple-choice options (2-10 options)
Vote restriction - One vote per user using session tracking
Real-time results - Live vote counting with percentage calculations
Poll expiry -  Set automatic expiration dates for time-sensitive polls
Responsive design - Works seamlessly on desktop and mobile devices

Installation
Prerequisites
Node.js (version 18 or higher)
npm

**Backend Setup**
Navigate to server directory
cd server
Install dependencies
npm install
Start the server
npm run dev
Server will run on** http://localhost:5000**

**Frontend Setup**
Open new terminal and navigate to client directory
cd client
Install dependencies
npm install
Start development server
npm start
Frontend will run on **http://localhost:3000**
